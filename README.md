# Project: Fisheye

## Context

This project was developed as part of my **OpenClassrooms training**.  
The goal was to practice JavaScript architecture using factories and to create an accessible interactive photo gallery.

---

## Technologies

- HTML5 & CSS3  
- JavaScript  
- Factories & modular JavaScript  
- DOM manipulation  
- Event handling  
- Accessibility  
- Responsive design  

---

## Project Description

Fisheye is a photography platform where users can discover photographer profiles and browse their media.  
Images and videos can be viewed inside an interactive lightbox, with a focus on usability and accessibility.
